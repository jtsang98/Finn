Finn Overview

Fairly Intelligent Non-humaN is a personal assistant that is voice commanded through an your phone. It will be able to handle day to day requests ranging from inquiries (check the weather) to home automation (lock the doors). Using LUIS AI and Google Speech to text, the application on your phone will turn speech to text and send queries to LUIS which will send JSON back to the phone. The application on the phone will then parse the JSON and, depending on the task, will send it to the arduino for it to execute the commands (turn on/off the lights etc.). Android Code
MainActivity.java

The main activity is the Java activity for the Android Application that controls the Arduino. The main activity will handle most of the applicatons function from managing the user interface of the application, connecting to the HC-06 bluetooth module, taking in speech and transcribing it to text, parse JSON response, sending intents to the Arduino, recieving messages back from the Arduino, and changing to message from the Arduino to speech. The code for MainActivity can be found here.

HttpHandler.java

This Java class provides the functions to send a query to Language Understanding (LUIS) and to build the JSON string that LUIS responds with. This class is called within the main activity and the JSON string is parsed within main activity as well. The code for HttpHandler can be found here.

How does this all work?
It's an extremely linear progress, but basically it goes like this:
  • You connect to the from the Android to the Arduino via the HC-06 bluetooth module.
  • Use Googe's Speech to Text to convert your speech to a string variable.
  • Use the string variable as a query to LUIS, and parse the JSON response.
  • Send the intent that your parsed from the JSON response to the Arduino via bluetooth.
  • Depedning on the intent, the Arduino will execute specific commands and send response back to the Andorid.
  • The Android will then convert the text back to speech.
Why is this impressive?
From a hardware standpoint completely understand. LED? Motors? Reading the temperature? Pretty weak stuff right? Well here is where your creativity comes in! The great thing about Finn is that's it's super robust!
  • Addition functions you want? Done, build a library and train it with phrases on LUIS.
  • Existing project you want to integrate? Easy, modify the library and come up with phrases!
  • Used a phrase you didn't teach Finn? Completely fine, as long as you trained it well.
